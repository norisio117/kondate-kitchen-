# 🍳 こんだてキッチン — 使い方

**1週間の献立づくり＆レシピ集アプリ**です。子育て家庭（取り分け・離乳食・ダイエット対応）でも、
**ひとり暮らし**（作り置き・お弁当・1人分）でも使えます。
和・洋・イタリアン・中華・韓国・エスニックまで、**全500レシピ＋朝食12品**を収録。

## 開き方
- `index.html` をダブルクリックするだけ（インターネット不要・オフラインで動きます）。
- データは端末内（ブラウザ）に自動保存。閉じても献立・買い物チェックは残ります。

## 5つの画面（下のタブで切替）

### 📅 献立
- 一番上の「**🍽 今日の気分で選ぶ**」…食べたいジャンル（和・洋・伊・中・韓・エスニック）をタップすると、
  その気分の主菜を1品パッと提案します。「**🔄 別の**」で別候補に。今夜のメインをサッと決めたい時に便利。
- 「**1週間の献立を作る**」を押すと、月〜日の夕食（主菜・副菜・汁物・ごはん）が自動で並びます。
- 「**👤 ひとり暮らしモード**」ON …
  - 人数が**1人分**になり、作り置き・お弁当・時短・ヘルシーなレシピが出やすくなります。
  - 取り分け・離乳食（月齢ガイド）の表示が消え、ひとり暮らし向けの「作り方・保存メモ」表示に切り替わります。
  - 家族向けに戻したい時はOFFにするだけ（設定は端末ごとに保存）。
- 「**🎯 好みに合わせる**」ON（初期状態）… レシピ集で **❤️お気に入り** を登録していくと、
  その**好きなジャンルや傾向**（例：イタリアンをよく登録→献立にイタリアンが増える）を毎週の献立に自動で反映します。
  使うほどその人好みに寄っていくので、**同じ端末で長く使うほど自分仕様**になります。設定の下に学習状況（例：「❤️5件から〇〇の傾向を反映中」）が表示されます。
- 各料理の **🔄** ボタンでその1品だけ作り直し。
- **🔒 固定**：気に入った料理の🔓を押すと🔒になり、もう一度「1週間の献立を作る」を押しても**その料理は残ります**（他だけシャッフル）。「月曜の魚は気に入ったから残して、あとは作り直す」が可能。
- 料理名の下「**👶 取り分けメモ・作り方**」をタップ → 離乳食/幼児への取り分け方、**📝調理手順**、アレルギー推定が開きます。
- 各料理には内容に応じた **絵文字アイコン**（🐟🍗🍛🥗 など）が付きます。
- 「**🥗 ダイエット配慮**」ON … カロリー控えめ・野菜多めの料理を優先し、副菜を増やします。
- 「**💰 節約重視**」ON … もやし・豆腐・鶏むね・缶詰など安い食材の料理を優先します。
- 「**🔁 前回と被らせない**」ON … 直近2週間に出た料理を避けて、毎週なるべく違うメニューにします。
- 「**👨‍👩‍👧‍👦 何人分で作る**」… 1〜8人分から選ぶと、**買い物リストの分量と食費が自動換算**されます（基準は4人分）。ひとり暮らしは1人分でどうぞ。
- 「**🙅 苦手な食材**」… 食材名を入れると（例：ピーマン しいたけ）、それを使う料理を献立・朝食に出しません。
- 「**💾 データのバックアップ**」… 設定・お気に入り・献立を書き出し/読み込みできます（機種変更や保存に）。
- 「**🚫 アレルギー除外**」… 卵・乳・小麦・えび・かに・そば・落花生・大豆から選ぶと、
  その食材を含む料理を献立・朝食に出しません。
  - ⚠️ 判定は**材料名からの簡易推定**です。調味料の微量成分や製品差・混入までは見ていません。
    **重いアレルギーは必ず実際の原材料表示と医師の指示を確認**してください。

### 🌅 朝食
- 「**1週間の朝食を提案**」で、忙しい朝でも作れる朝ごはんを月〜日でシャッフル。各日 **🔄** で作り直し。
- 取り分けメモ付き。アレルギー除外も効きます。（※朝食は買い物リストには含めません）

### 🛒 買い物
- 今週の献立から必要な食材を **自動で合算**（例：にんじん 6本）。
- 売り場ごと（野菜・肉魚・調味料…）に分類。タップでチェック → スーパーでそのまま使えます。
- 上部に **💰 食材費の目安**（4人×7日の夕食分・1日あたり・1人1食あたり）を表示。
  ※ 地域・店・時期で変わるおおよその目安です。
- **📋 コピー**：買い物リストを文字でコピー → LINEのメモや家族への共有にそのまま貼れます。
- **🖨 印刷**：買い物リストをきれいに印刷（不要なボタン等は印刷されません）。

### 🍀 栄養
- 1日ごとに「主食・主菜・副菜・汁物・野菜・魚」が揃っているかを色付きで表示。
- カロリーの目安（控えめ🟢/標準🟡/高め🔴）と、週まとめ（平均カロリー・魚の日数）も。

### 📖 レシピ（全500品＋朝食12品・全品に📝調理手順つき）
- 収録レシピ一覧（主菜・副菜・汁物・ワンプレート）。各レシピに**絵文字アイコン・アレルギー推定バッジ・💰節約マーク**付き。
- 「材料・作り方・取り分けを見る」をタップ → 材料、**📝調理手順**、取り分け／保存メモが開きます。
- 右上の **♡** を押すと**お気に入り登録**。お気に入りは献立に出やすくなり、「❤️お気に入り」フィルタでまとめて見られます。
- **ジャンルで絞り込み**：「🍚和」「🍞洋」「🍝伊(イタリアン)」「🥟中(中華)」「🌶️韓(韓国)」「🍛エスニック」。
- ほかに「❤️お気に入り」「🐟魚」「🍱作り置き」「🍙お弁当」「💰節約」「🔥バズり」「🕒時短」「ヘルシー」などで絞り込み。
- アレルギー除外を設定すると、該当レシピは「除外中」と表示され薄くなります。
- 🧊 **冷蔵庫の余り物で探す**：上部の入力欄に手持ちの食材を入れる（例：`なす 豚 たまご`）と、
  それを使うレシピを「○個一致」の多い順に表示。余り物の使い切り・献立決めに便利です。
- 🍳 **この食材で献立を組む**：同じ入力欄に食材を入れて青いボタンを押すと、
  その食材を**いちばん多く使える組み合わせ**（主菜＋副菜＋汁物、ワンプレート時は主菜＋副菜）を提案します。
  「入力5種のうち4種を使用（未使用：〇〇）」のように**使い切り度**も表示。「🔄 組み直す」で別の組み合わせに。
- **🕒時短・市販の素**：永谷園・丸美屋・味の素 Cook Do・ハウス・キッコーマン・S&B・ミツカン・
  カゴメ・ニチレイ・マ・マー・創味・ヤマモリ・ハチ食品などの実在商品を使った時短レシピを収録。
  商品名は買い物リストにもそのまま並びます。
  - ※ 市販の素は塩分・油・唐辛子を含むものが多く、**離乳食には不向き**。取り分けメモに「離乳食NG／幼児は少量」を明記。
- **🔥バズり**：ネット/SNSで話題の家庭ごはん（無限ピーマン・やみつき塩キャベツ・サラダチキン・
  悪魔のおにぎり・無水トマトカレー・サバ缶トマト煮など）。ダイエット向きの低カロリーも多数。

## iPhoneのホーム画面に置く（アプリ風に・通年使う）
1. Safariで `index.html` を開く（または下記の公開URL）。
2. 共有ボタン → 「ホーム画面に追加」。
3. アイコン🍳をタップすれば、アプリのように起動します。

> 💡 **長く（通年）使うコツ**：献立・お気に入り・好み学習・設定は、その端末のブラウザに保存され、
> 同じURLを開けば毎回そのまま復元されます。ずっと自分仕様で使い続けられます。
> ただし iPhone のSafariは「たまにしか開かないサイト」のデータを一定期間で消すことがあります。
> **「ホーム画面に追加」して使えばこの心配がなくなり**、データが長期間保持されます。
> 機種変更や念のためのバックアップには、献立ページの「💾 データのバックアップ（書き出し／読み込み）」を使ってください。

## レシピを追加・変更したいとき
- `recipes.js` を編集すると料理を増やせます（コメントに書き方の説明あり）。
- 「○○のレシピを追加して」と司馬懿に言うだけでもOK。
- 🤖 **自動追加**：毎月3日に司馬懿が**新レシピを6品自動で追加**します（品数の少ないジャンルを優先・重複なし・構文チェック済み）。
  - ※ URLで共有している場合、公開サイトは自動更新されません。増えた分を同僚に届けるには、
    フォルダを再度 Netlify Drop にドロップして更新してください。頻度や品数を変えたい時は司馬懿に言ってください。

## ファイル構成
| ファイル | 役割 |
|---|---|
| `index.html` | アプリ本体（画面・献立生成・買い物・栄養のロジック） |
| `recipes.js` | レシピのデータベース（料理・材料・カロリー・取り分けメモ） |
| `manifest.json` / `service-worker.js` | ホーム画面追加・オフライン動作用 |
| `icon-192.png` / `icon-512.png` | アプリアイコン |

> メモ：分量は「4人分（大人2＋子ども2）」基準。カロリーは1人分のざっくり目安です。
